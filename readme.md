# Kotlin Rail Architecture Simulator

This project provides a simulator, assembler, and terminal UI for the simple 8-bit Rail architecture.

While not intended for serious applications, this project serves as a hands-on introduction to assembly language and hardware simulation.

Being new to Kotlin, the code may not fully adhere to idiomatic Kotlin practices.
